Ce que contient un cahier des charges classique pour un projet informatique:
- Description du projet
- Présentation de l'entreprise (pas obligatoire)
- Back End :
	- Diagramme Use Cases
	- Tableau décrivant les fonctionnalités
	- Quelques scénarios
- Front End :
	- Charte graphique
	- Diagramme de navigabilité
	- Quelques pages
- Budget
- Délivrable (durée du projet, type de méthode de livrable etc)
